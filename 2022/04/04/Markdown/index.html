Markdown

Posted on 2022-04-04

more reading : https://markdown.com.cn/basic-syntax/

Basic

Elements Markdown Syntax
标题（Heading） # H1
## H2
### H3
粗体（Bold） *bold text*
斜体（Italic） *italicized text*
引用块（Blockquote） > blockquote
有序列表（Ordered List） 1. First item
2. Second item
3. Third item
无序列表（Unordered List） - First item
- Second item
- Third item
代码（Code） `code`
分隔线（Horizontal Rule） -–
链接（Link） [title](https://www.example.com)
图片（Image） !alt text

Extension

Elements Markdown Syntax
表格（Table）
代码块（Fenced Code Block） ```
{
“firstName”: “John”,
“lastName”: “Smith”,
“age”: 25
}
```
脚注（Footnote） Here’s a sentence with a footnote. [^1]
[^1]: This is the footnote.
定义列表（Definition List） term
: definition
删除线（Strikethrough） ~~The world is flat.~~
任务列表（Task List） - [x] Write the press release
- [ ] Update the website
- [ ] Contact the media
